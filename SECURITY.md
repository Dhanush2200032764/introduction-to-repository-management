# Mergington High School Security Policy

## Reporting a Vulnerability

At Mergington High, we take the security of our Extra-Curricular Activities website seriously.

If you discover a vulnerability:

1. Do NOT create public issue
2. Go to Security tab
3. Click "Report a vulnerability"
4. Fill details:
   - Description
   - Steps
   - Impact
   - Fix (if possible)

5. Email: techsupport@mergingtonhigh.example.edu

## Response Timeline

- 2 days acknowledgement
- 5 days assessment
- Critical issues fixed immediately

## Thank You

Thanks for helping secure our system.
